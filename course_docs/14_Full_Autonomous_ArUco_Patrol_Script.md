# Module 14: Full Autonomous ArUco Patrol Script

## Goal

In this module, students bring the full project together into one working autonomous ArUco patrol script.

The final script combines:

- Go2 WebRTC connection
- Go2 camera stream
- OpenCV ArUco detection
- marker-controlled actions
- marker confirmation
- cooldowns
- built-in obstacle avoidance
- patrol behavior
- wall and corner recovery
