# Course Overview: Autonomous Unitree Go2 Control with ArUco Markers and OpenCV

## Course Title

**Autonomous Unitree Go2 Control with ArUco Markers and OpenCV**

## Big Idea

In this course, students will learn how to use computer vision to control a Unitree Go2 robot dog with printed ArUco markers.

The main idea is:

> ArUco markers act like visual command signs that the robot can read through its camera.

Instead of only controlling the Go2 with a joystick, phone app, or remote controller, students will create printed markers that the robot can recognize. Each marker has an ID number, and each ID can be connected to a robot behavior.

Example:

| Marker ID | Robot Behavior |
|---|---|
| 0 | Stop |
| 1 | Stand up |
| 2 | Sit |
| 3 | Move forward |
| 4 | Turn or search |

Later in the course, the robot will move autonomously, patrol an area, scan for markers, and react when it finds them.
